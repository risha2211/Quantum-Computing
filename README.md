# Implemented Quantum Algorithms

## Deutsch Jozsa Algorithm
Determines whether a function is constant or balanced using quantum parallelism.

## Bernstein Vazirani Algorithm
Finds a hidden binary string using a single oracle query.

## Simon's Algorithm
Demonstrates exponential quantum speedup for identifying hidden periodicity.

## Grover's Search Algorithm
Performs amplitude amplification for quadratic speedup in unsorted search problems.

## Shor's Algorithm
Explores quantum factorization and its implications for modern cryptography.

## Quantum Phase Estimation
Estimates eigenvalues of unitary operators and forms the basis of several quantum algorithms.

## Quantum Teleportation
Simulates transfer of quantum state information using entanglement and classical communication.

# Quantum Cryptography

## BB84 Protocol
Implementation of quantum key distribution using polarized qubit states.

## E91 Protocol
Entanglement based quantum key distribution using Bell pairs.
